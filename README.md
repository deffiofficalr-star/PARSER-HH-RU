# 📊 PARSER-HH-RU

**Парсер вакансий с hh.ru**

---

## 📌 Что это?

PARSER-HH-RU — это программа для парсинга вакансий с сайта hh.ru. Позволяет искать вакансии по ключевым словам, региону и опыту работы, собирать информацию о зарплатах, навыках и компаниях, а также сохранять данные в CSV и JSON.

---

## ✨ Возможности

- 🔍 Поиск вакансий по ключевому слову
- 📍 Выбор региона (Москва, СПБ, Россия)
- 💼 Фильтр по опыту работы
- 📊 Статистика по компаниям, городам, навыкам
- 💰 Средняя зарплата
- 📁 Сохранение в CSV и JSON
- 🏆 Топ-10 компаний и городов

---

```bash
git clone https://github.com/deffiofficalr-star/PARSER-HH-RU.git
